# Casino Frontend

Aplicación web para jugadores de Royal Casino. Interfaz elegante con lobby de juegos y tres juegos jugables.

## Inicio rápido

```bash
npm install
npm run dev
```

Abre `http://localhost:5173`

> Requiere que **casino-api** esté corriendo en el puerto 3001.

## Características

- Registro de jugador con bono de bienvenida ($1,000)
- Lobby con catálogo de juegos
- Tragamonedas, Blackjack y Ruleta
- Depósitos y saldo en tiempo real
- Diseño oscuro con estética de casino premium
