# Casino Admin

Panel de administración para monitorear la plataforma de casino en tiempo real.

## Inicio rápido

```bash
npm install
npm run dev
```

Abre `http://localhost:5174`

> Requiere que **casino-api** esté corriendo en el puerto 3001.

## Secciones

- **Dashboard** — KPIs: usuarios, partidas, apuestas, pagos y ventaja de la casa
- **Usuarios** — Lista de jugadores registrados con saldo
- **Partidas** — Historial de juegos con resultados
- **Transacciones** — Depósitos, retiros, apuestas y premios

Los datos se actualizan automáticamente cada 10 segundos.
